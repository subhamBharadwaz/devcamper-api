# DevCamper API

> Backend API for DevCamper application, which is bootcamp directory website

## usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your owm.

# Install Dependencies

```
npm install

```

## Run App

```
# Run in dev mode
npm run dev

#Run in prod mode
npm start

```
